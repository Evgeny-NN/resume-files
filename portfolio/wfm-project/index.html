Проект WFM – Автоматизация планирования ресурсов

Описание: Внедрение системы Workforce Management (WFM) для автоматизации планирования, прогнозирования и мониторинга работы контакт-центров.

Моя роль: Системный аналитик (проектирование архитектуры, разработка ADR, интеграционные схемы).

Архитектурные решения (ADR)

  • ADR-001: Внедрение системы WFM C4 контекст (контекст, интеграции, ролевая модель)

Другие артефакты

Здесь будут появляться BRD, BPMN-схемы, технические задания и другие документы.

Все артефакты обезличены для портфолио. Оригиналы содержат коммерческую информацию.
