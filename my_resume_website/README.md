> **Персональный сайт-резюме системного аналитика**  
> Демонстрирует опыт проектирования интеграций, работу с требованиями, документацией (BRD, PRD, ADD), а также ключевые hard skills. Сайт адаптивен, оптимизирован для рекрутеров и поисковых систем.

---

## 🎯 Для кого этот сайт

- **HR и технические рекрутеры** — быстрый доступ к опыту, проектам и технологическому стеку.
- **Руководители IT-отделов** — оценка компетенций в системном анализе, интеграциях, архитектуре.
- **Коллеги-аналитики** — пример оформления портфолио в виде живого сайта.

---

## ✨ Ключевые особенности (с точки зрения системного аналитика)

- **Структурированное представление опыта**  
  Проекты разбиты с указанием роли, задач и достижений — легко читается и масштабируется.

- **Визуальное выделение hard skills**  
  Категории навыков (документация, системный анализ, проектирование, тестирование, мониторинг) оформлены в виде «бейджей» с анимацией.

- **Готовая микроразметка Schema.org**  
  Повышает шансы, что поисковые системы покажут ваше резюме в расширенном виде (с фото, контактами, опытом).

- **Защита email от спам-ботов**  
  Email собирается через JavaScript — снижает количество нежелательных писем.

- **Кнопка «Скачать PDF»**  
  Рекрутеры могут одним кликом сохранить резюме в формате PDF для внутренних систем.

- **Плавная навигация по разделам**  
  Опыт, навыки, образование, о себе — быстрый доступ к нужному блоку.

- **Адаптивность и доступность**  
  Корректно отображается на любых устройствах, соответствует контрастности WCAG.

## 🛠️ Технологии, которые использованы в разработке сайта

| Технология | Для чего |
|------------|----------|
| **HTML5** | Семантическая вёрстка, микроразметка |
| **CSS3** | Flexbox/Grid для сеток, медиазапросы, анимации hover |
| **JavaScript (ES6)** | Защита email, плавный скролл, кнопка печати PDF |
| **Git & GitHub** | Контроль версий, хостинг через GitHub Pages |
| **Figma (опционально)** | Прототип макета перед вёрсткой |

> Эти технологии не имеют прямого отношения к моей работе системного аналитика, но демонстрируют владение инструментами для создания clean code и презентационных материалов.
