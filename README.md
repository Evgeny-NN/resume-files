# resume-files

Этот репозиторий содержит мои профессиональные артефакты: резюме, портфолио проектов, сертификаты.

## Структура

- `/resume/` — основное резюме и сопутствующие файлы (сертификаты, изображения)
- `/portfolio/` — портфолио проектов с детальным описанием и артефактами
  - `/portfolio/openapi/` — OpenAPI-спецификации REST API
